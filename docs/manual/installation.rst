************
Installation
************

You can install ABlog using `pip <https://pip.pypa.io/en/stable/>`__:

.. code-block:: bash

   $ pip install -U ablog

or `conda-forge <https://github.com/conda-forge/miniforge>`__:

.. code-block:: bash

   $ conda install ablog

This will install ABlog and its dependencies.
